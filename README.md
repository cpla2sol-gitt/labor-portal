# 노무 서식 포털 — HR Document Portal

> 채용부터 퇴직까지, 노무관리에 필요한 서식을 온라인으로 작성하고 바로 PDF로 출력하는 웹 포털입니다.  
> 결재라인·회사 정보를 한 번만 설정하면 전체 서식에 자동 반영됩니다.

🔗 **배포 주소:** `https://cpla2sol-gitt.github.io/labor-portal/`

---

## 📁 파일 구성

```
labor-portal/
├── index.html    — 메인 페이지 (HTML 구조 + 편집기 패널 UI)
├── style.css     — 전체 스타일 (CSS 변수 기반, A4 출력 포함)
├── forms.js      — 35종 서식 HTML 정의
├── app.js        — 네비게이션 · 결재란 · 회사 정보 설정 로직
└── editor.js     — 실시간 디자인 편집기 (색상·크기·로고)
```

## ⚙ 주요 기능

### 회사 정보 일괄 설정
우상단 **⚙ 설정** 버튼에서 회사명·대표자명·사업장 주소를 한 번만 입력하면 전체 서식에 자동 반영됩니다. 자문사(고객사)가 바뀔 때마다 여기서만 변경하면 됩니다.

### 결재라인 설정
1단계(대표) ~ 4단계(담당→팀장→본부장→대표) 중 선택 가능합니다. 설정에서 기본값을 지정하거나 각 서식에서 개별 변경도 가능합니다. 증명서류(재직증명서·퇴직증명원 등)는 결재란이 자동으로 숨겨집니다.

### PDF 출력
- 서식 툴바의 **[PDF 출력]** — 현재 서식 1장 출력
- 헤더의 **[전체 출력]** — 전체 서식을 A4 한 장씩 일괄 출력

### 실시간 디자인 편집기
헤더의 **🎨 디자인** 버튼으로 열 수 있습니다.

| 항목 | 편집 가능 내용 |
|------|----------------|
| 색상 테마 | 헤더·사이드바·강조색·배경·테이블 셀·결재란 / 5종 프리셋 제공 |
| 로고 | PNG·JPG·SVG 업로드, 사이트명, 부제목 변경 |
| 서식 크기 | 본문 폰트(7~13pt), 제목 폰트, 셀 높이, 서명란 너비 |
| A4 여백 | 상하(10~40mm), 좌우(10~35mm) |
| CSS 내보내기 | 현재 설정 → `style.css` 직접 수정용 코드로 복사 |

## 🔧 코드 수정 가이드

### 서식 추가하기
**1단계** — `forms.js`의 `FORMS` 객체에 새 항목 추가
```js
'my-form': {
  title: '서식 이름',
  sub: '근거 법령',
  hasApproval: true,    // true: 결재란 있음 / false: 없음
  notice: '상단 공지',  // 빈 문자열이면 공지 숨김
  html: () => `...`     // 서식 HTML 반환 함수
}
```

**2단계** — `index.html` 사이드바에 메뉴 항목 추가
```html
<div class="sb-item" id="m-my-form" onclick="go('my-form')">
  <span class="n">36</span>새 서식 이름
</div>
```

### 색상·크기 코드로 수정하기
`style.css` 상단 `:root {}` 블록의 CSS 변수를 직접 수정합니다.

```css
:root {
  --navy-dk:    #0d2137;  /* 헤더·버튼 배경색 */
  --navy:       #1b3a5c;  /* 사이드바 배경색 */
  --gold:       #c8a354;  /* 강조 색상 */
  --form-font:  9.5pt;    /* 서식 기본 폰트 크기 */
  --title-font: 18pt;     /* 서식 제목 폰트 크기 */
  --cell-pad-v: 4pt;      /* 테이블 셀 높이 (상하 패딩) */
  --sign-w:     150pt;    /* 서명란 너비 */
  --a4-pad-v:   28mm;     /* A4 상하 여백 */
  --a4-pad-h:   22mm;     /* A4 좌우 여백 */
}
```

### 결재란 직급 변경하기
`app.js`의 `AP_LABELS` 객체를 수정합니다.
```js
const AP_LABELS = {
  1: ['대표'],
  2: ['팀장', '대표'],
  3: ['담당', '팀장', '대표'],
  4: ['담당', '팀장', '본부장', '대표']
};
```
